# ChatGPT Telegram Bot

## Описание

Этот телеграм-бот интегрируется с API ChatGPT для генерации текста по запросу пользователя.
Бот способен поддерживать диалог, сохраняя контекст предыдущих сообщений.

## Возможности

-   Ответы на команды `/start` и `/help`.
-   Генерация текста с использованием ChatGPT на любое текстовое сообщение.
-   Сохранение истории диалога для предоставления контекстно-зависимых ответов.
-   Сброс контекста диалога с помощью команды `/start` или кнопки "Новый запрос".

## Установка

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/github-main-user/chatgpt-telegram.git
    cd chatgpt-telegram
    ```

2.  **Установите зависимости:**
    ```bash
    uv sync
    ```

## Конфигурация

1.  **Переименуйте `.env.example` в `.env`:**
    ```bash
    mv .env.example .env
    ```

2.  **Добавьте ваши токены в файл `.env`:**
    -   `TELEGRAM_BOT_TOKEN`: Токен вашего телеграм-бота.
    -   `OPENROUTER_API_KEY`: Ваш API ключ от OpenRouter.

## Использование

Запустите бота:
```bash
uv run python main.py
```
